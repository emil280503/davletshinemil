# PythonProject2

Краулер на Python, который скачивает HTML-страницы одного языка, сохраняет их в файлы без очистки разметки и формирует `index.txt`.

## Что делает

- обходит страницы с русской Википедии;
- сохраняет минимум 100 HTML-страниц;
- пишет соответствие `номер файла -> ссылка` в `index.txt`.

## Запуск

```bash
python main.py --limit 100 --output downloaded_pages
```

После запуска в каталоге `downloaded_pages` будут:

- `001.html`, `002.html`, ...;
- `index.txt`.

## Сдача

Для архива с выгрузкой можно упаковать каталог `downloaded_pages` в ZIP.
