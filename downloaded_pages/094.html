ISO 3166-1 — часть стандарта ISO 3166, поддерживаемого агентством по стандарту ISO 3166. ISO 3166-1 содержит коды стран и территорий и определяет три разных кода для каждой страны или территории:

  • ISO 3166-1 numeric, трёхцифровые коды, идентичные статистическим кодам, используемым в ООН.
  • ISO 3166-1 alpha-2, двухбуквенные коды, имеющие множество применений, включая Интернет (обозначения для доменов первого уровня).
  • ISO 3166-1 alpha-3, трёхбуквенные коды.

ISO 3166-1 содержит коды 249 стран и территорий. Список кодов ISO 3166-1 синхронизирован со стандартом ООН Коды стран и территорий для использования в статистике (Стандарт M49 отдела статистики ООН) с одним исключением: ISO 3166-1 содержит коды для Китайской Республика (Тайвань), отсутствующие в «Стандарте M49» ООН.

В соответствии с правилами агентства по стандартам ISO 3166 новые коды могут вноситься в ISO 3166-1 только при условии появления соответствующих кодов в одном из двух источников:

  • Бюллетень ООН «Названия стран»
  • Коды стран и территорий для использования в статистике («Стандарт M49» отдела статистики ООН)

В свою очередь, «Стандарт M49» ООН составляется на основе бюллетеня «Названия стран» и других источников ООН. Для того, чтобы быть включённой в бюллетень «Названия стран», страна или территория должна удовлетворять любому из нижеперечисленных условий:

  • быть членом ООН
  • быть членом любого из специализированных агентств ООН
  • принимать участие в составлении свода законов Международного Суда ООН.

Страна или регион получает новый код при смене названия, а числовой код изменяется при изменении границ. Некоторые коды зарезервированы для различных целей, например зарезервированными становятся устаревшие коды либо коды субъектов, не входящих в ISO 3166-1. Например, поскольку Европейский союз не является страной, он не включен в ISO 3166-1, но по практическим причинам код Европейского союза EU зарезервирован для его идентификации.
